## 摘要
增量式开放词汇3D实例-语义建图对于在复杂日常环境中运行的自主智能体至关重要。然而，由于需要鲁棒的实例分割、实时处理以及灵活的开放集推理，这仍然具有挑战性。现有方法通常依赖于闭集假设或密集的逐像素语言融合，这限制了可扩展性和时间一致性。我们提出了OVI-MAP，它将实例重建与语义推理解耦。我们建议构建一个与类别无关的3D实例地图，该地图从RGB-D输入中增量构建，而语义特征仅使用视觉-语言模型从一小组自动选择的视图中提取。这种设计使得在整个在线探索过程中能够实现稳定的实例跟踪和零样本语义标注。我们的系统能够实时运行，并在标准基准测试中优于最先进的开放词汇建图基线方法。

## Abstract
Incremental open-vocabulary 3D instance-semantic mapping is essential for autonomous agents operating in complex everyday environments. However, it remains challenging due to the need for robust instance segmentation, real-time processing, and flexible open-set reasoning. Existing methods often rely on the closed-set assumption or dense per-pixel language fusion, which limits scalability and temporal consistency. We introduce OVI-MAP that decouples instance reconstruction from semantic inference. We propose to build a class-agnostic 3D instance map that is incrementally constructed from RGB-D input, while semantic features are extracted only from a small set of automatically selected views using vision-language models. This design enables stable instance tracking and zero-shot semantic labeling throughout online exploration. Our system operates in real time and outperforms state-of-the-art open-vocabulary mapping baselines on standard benchmarks.

*   **研究动机与核心问题**：现有方法在实现**开放词汇（open-vocabulary）** 3D建图时面临三大挑战：
    1.  **存储与计算开销大**：直接将视觉语言模型（VLM）提取的高维、连续语义特征存储在密集的体素网格中，会导致巨大的内存和计算负担。

    3.  **时序一致性差**：在没有稳定3D实例的情况下，跨帧聚合逐像素的开放集特征会受到遮挡、视角变化和2D分割噪声的影响。
*   **整体含义**：本文旨在解决上述挑战，提出一种能够**实时在线运行、支持开放词汇查询的增量式3D实例-语义建图系统（OVI-MAP）**。

## 2. 方法论
*   **核心思想（解耦设计）**：将“实例重建”与“语义推理”两个过程解耦。
    1.  首先，基于纯几何和区域一致性证据，在线构建一个稳定的、与类别无关的3D实例地图。
    2.  然后，仅在观察到对物体外观具有信息量的新视角时，才利用VLM提取并聚合该物体的语义特征。
*   **关键技术细节**
    *   **类无关实例地图重建 (Class-Agnostic Instance Map Reconstruction)**：
        *   **输入**: RGB-D序列及相机位姿。
        *   **2D实体分割与几何细化**: 使用CropFormer生成类无关实体提议掩码；利用深度不连续性进行几何分割以细化掩码边界，减少过合并错误。
        *   **3D提升与关联**: 将精炼后的2D掩码通过深度和位姿提升为3D点云块。通过空间投票机制（统计点云块覆盖的体素中已有的实例标签），决定其是归属于现有全局实例还是创建新实例。整个过程不依赖任何语义信息。
        *   **增量融合与标签稳定化**: 使用TSDF体素网格融合几何。每个体素通过累计投票数来稳定其所属的全局实例ID。通过合并空间邻近的超点来处理过分割问题。
    *   **视图自适应的语义特征聚合 (View-Adaptive Semantic Feature Aggregation)**：
        *   **目标**: 高效地为每个稳定的3D实例生成鲁棒的开放集语义嵌入向量。
        *   **对象中心视图选择 (Object-Centric View Selection)**：
            *   为每个3D实例维护一个单位球面覆盖图（spherical coverage map），记录已从哪些观察方向看过该物体。
            *   当处理新帧时，计算当前视角能带来的“新覆盖区域”比例。若比例超过阈值 `θ_novel` ，则认为该视角提供了新的外观信息，将其选为用于提取特征的“关键视角”。
            *   相比基线方法（如OpenMask3D采用的“可见像素计数”策略），该方法能主动选择多样化的视角，避免冗余的前向视图，从而用更少的VLM查询获得更全面的物体表征。
        *   **特征提取与融合**: 对于选中的关键帧，对物体进行两种裁剪（带背景的原图裁剪、去除背景的掩码裁剪），输入VLM（如SigLIP）提取视觉特征。采用基于可见像素数量的加权平均方式对多次观测的特征进行增量融合。
