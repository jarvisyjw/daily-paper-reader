## 摘要
并行计算与GPU加速的最新进展为计算密集型学习问题（如主动SLAM——通过选择动作来减少不确定性并改进联合建图与定位）创造了新的机遇。然而，现有的基于深度强化学习的方法仍受限于缺乏可扩展的并行训练。在本工作中，我们通过提出一个用于主动SLAM的可扩展端到端深度强化学习框架来解决这一挑战，该框架支持大规模并行训练。与现有技术相比，我们的方法显著减少了训练时间，支持连续动作空间，并有助于探索更现实的场景。本框架已作为开源项目发布，以促进可复现性和社区采用。

## Abstract
Recent advances in parallel computing and GPU acceleration have created new opportunities for computation-intensive learning problems such as Active SLAM -- where actions are selected to reduce uncertainty and improve joint mapping and localization. However, existing DRL-based approaches remain constrained by the lack of scalable parallel training. In this work, we address this challenge by proposing a scalable end-to-end DRL framework for Active SLAM that enables massively parallel training. Compared with the state of the art, our method significantly reduces training time, supports continuous action spaces and facilitates the exploration of more realistic scenarios. It is released as an open-source framework to promote reproducibility and community adoption.

## **论文总结：面向主动SLAM的大规模并行深度强化学习**

### **1. 核心问题与整体含义**
*   **研究背景**：主动SLAM（ASLAM）旨在控制机器人在执行SLAM的同时，通过决策来减少位姿和地图的不确定性并探索未知环境。深度强化学习（DRL）因其处理部分可观测性和高维感官输入的能力而成为有前景的解决方案。
*   **核心问题**：现有基于DRL的ASLAM方法面临两大瓶颈：
    1.  **训练时间极长**（例如超过60小时），严重阻碍了实际应用。
    2.  **假设过于简化**，如使用离散动作空间和简化的二维环境，限制了在真实机器人场景中的适用性。
*   **整体目标**：利用现代GPU并行计算能力，构建一个可扩展的、端到端的DRL框架，以大幅缩短ASLAM策略的训练时间，并支持连续动作空间和更真实的仿真环境。

### **2. 方法论**
*   **核心思想**：将ASLAM建模为一个部分可观测马尔可夫决策过程（POMDP），并利用大规模并行仿真（同时训练数百个智能体）来加速策略学习。框架包含一个轻量级的、GPU向量化的SLAM后端来提供实时的位姿不确定性估计，并将其集成到DRL的观察和奖励函数中。
*   **关键技术细节**：
    *   **GPU加速的固定滞后SLAM骨干网络**：采用轻量级固定滞后位姿图优化方案，舍弃全局回环检测以保证计算确定性和实时性。通过高斯-牛顿法求解局部图优化，并从Hessian矩阵中提取当前位姿协方差作为不确定性标量信号 `Ut`。整个流程在GPU上批量、向量化执行。
    *   **观察与动作空间**：
        *   观察 `ot`: [128维LiDAR测距值, 当前偏航角 `yaw`, 不确定性标量 `Ut`]。
        *   动作 `at`: 连续的线速度 `v` (0, 0.4) m/s 和角速度 `ω` (-1, 1) rad/s。
    *   **不确定性感知的奖励函数设计**：奖励函数精心组合了三个目标：
        1.  **探索奖励 (`R_explore`)**: 鼓励访问新区域。其内在缩放因子 `I` 会根据当前不确定性水平动态调整：低不确定时直接鼓励探索；高不确定时，只有能降低不确定性的探索行为才会获得奖励。这实现了探索与重定位行为之间的平滑切换。
        2.  **绝对不确定性惩罚 (`R_abs`)**: 指数形式惩罚高不确定性状态。
        3.  **微分不确定性奖励 (`R_Δ`)**: 奖励不确定性降低的行为。
        4.  **(可选)防贴墙正则化**: 通过调整靠近障碍物时的奖励权重，防止智能体产生“贴墙走”的病态行为。
    *   **策略优化与架构**：使用近端策略优化算法训练一个包含GRU层的循环神经网络策略。GRU层使智能体能够捕捉不确定性的时间动态（如上升或下降趋势）。
    *   **SLAM无关的微调桥接器**：为了解决将训练好的策略迁移到不同SLAM系统时因不确定性信号分布变化导致的性能下降问题，提出了一个训练桥接器。它通过在微调阶段逐步混合新旧SLAM系统的不确定性信号 (`U_bridge_t = α_t * U_new_t + (1-α_t) * U_old_t`)，实现平滑过渡。
