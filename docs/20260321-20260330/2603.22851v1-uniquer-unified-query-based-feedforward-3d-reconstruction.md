## 摘要
我们提出了UniQueR，一个基于查询的统一前馈框架，用于从无位姿图像中进行高效且准确的三维重建。现有的前馈模型（如DUSt3R、VGGT和AnySplat）通常预测逐像素的点云图或像素对齐的高斯分布，这些方法本质上仍是2.5维的，且仅限于可见表面。相比之下，UniQueR将重建问题表述为一个稀疏的三维查询推理问题。我们的模型学习一组紧凑的三维锚点作为显式的几何查询，使网络能够推断场景结构（包括遮挡区域的几何信息）——仅需一次前向传播。每个查询直接在全局三维空间（而非逐帧相机空间）中编码空间和外观先验，并生成一组用于可微分渲染的三维高斯分布。通过利用多视图特征之间的统一查询交互以及解耦的交叉注意力设计，UniQueR在显著降低内存和计算成本的同时实现了强大的几何表达能力。在Mip-NeRF 360和VR-NeRF数据集上的实验表明，UniQueR在渲染质量和几何精度方面均优于最先进的前馈方法，且所使用的基元数量比密集替代方案少一个数量级。

## Abstract
We present UniQueR, a unified query-based feedforward framework for efficient and accurate 3D reconstruction from unposed images. Existing feedforward models such as DUSt3R, VGGT, and AnySplat typically predict per-pixel point maps or pixel-aligned Gaussians, which remain fundamentally 2.5D and limited to visible surfaces. In contrast, UniQueR formulates reconstruction as a sparse 3D query inference problem. Our model learns a compact set of 3D anchor points that act as explicit geometric queries, enabling the network to infer scene structure, including geometry in occluded regions--in a single forward pass. Each query encodes spatial and appearance priors directly in global 3D space (instead of per-frame camera space) and spawns a set of 3D Gaussians for differentiable rendering. By leveraging unified query interactions across multi-view features and a decoupled cross-attention design, UniQueR achieves strong geometric expressiveness while substantially reducing memory and computational cost. Experiments on Mip-NeRF 360 and VR-NeRF demonstrate that UniQueR surpasses state-of-the-art feedforward methods in both rendering quality and geometric accuracy, using an order of magnitude fewer primitives than dense alternatives.

### **论文核心问题与整体含义**

*   **研究动机**：传统的基于优化的三维重建方法（如NeRF、3DGS）虽然质量高，但计算成本高昂且无法泛化。近年来兴起的前馈式三维重建方法（如DUSt3R, AnySplat）能实现快速推理，但其核心是基于像素对齐的2.5D表示（如深度图、点云图），这导致其生成的几何体被“钉”在输入图像的可见表面上，**无法有效重建被遮挡或未被观察到的区域**，在新视角下容易出现空洞和伪影。
*   **整体含义**：本文旨在解决前馈式三维重建中“表示受限”的根本问题。作者提出了一种全新的范式——将三维重建视为一个**稀疏的三维查询推理问题**。通过引入一组可学习的、具有明确三维位置的“查询”，模型能够直接在全场景的三维空间中布局几何体，从而实现对可见和不可见区域的统一、完整建模。

### **论文提出的方法论**

*   **核心思想**：用一个稀疏的、可学习的**三维查询集合**作为场景的核心表示。每个查询是一个带有三维坐标和高维嵌入向量的锚点，它可以从多视图图像特征中聚合信息，并“孵化”出一组局部的高斯椭球体用于最终渲染。
*   **关键技术细节**：
    1.  **两阶段特征提取与几何先验获取**：
        *   使用DINOv2 ViT提取每帧图像特征。
        *   通过交替注意力Transformer聚合跨视图信息。
        *   解码出每帧的相机位姿、点云图和置信度图（继承自VGGT/Pi3框架），为后续查询提供粗略的几何先验。
    2.  **混合查询初始化策略**：
        *   **一半查询**从预测的点云图中采样初始化，确保覆盖观测到的表面。
        *   **另一半查询**在场景包围盒内随机初始化，鼓励探索和填充未观测区域。这种策略解决了纯随机初始化导致训练不稳定的问题。
    3.  **解耦注意力设计的高效查询传播**：
        *   为了高效处理大量图像特征Token和少量查询Token的交互，作者设计了“解耦注意力”模块（见图3）。
        *   首先进行从查询到图像特征的交叉注意力（`Cross-Attn`），让查询吸收图像信息。
        *   然后在查询内部进行自注意力（`Self-Attn`）。
        *   复杂度从 `O((Q+N*HW/p^2)^2)` 降至 `O(Q*N*HW/p^2 + Q^2)`，显著提升了效率和可扩展性。
    4.  **高斯孵化与渲染监督**：
        *   每个更新后的查询通过MLP预测一个位置偏移和K个高斯椭球体的局部属性（中心、尺度、旋转、不透明度、颜色）。
        *   使用可微分的3D高斯溅射将整个高斯集合渲染成RGB和深度图。
    5.  **关键训练策略——新视角监督**：
        *   监督视图是输入视图的超集。例如，给定2个输入视图，模型会在4个新视角下进行渲染并计算损失。
        *   **这是驱动模型填充遮挡区域的核心机制**：如果模型只重建输入可见部分，那么在新视角下就会出现空洞并被损失函数惩罚。
