## 摘要
视觉重定位是三维计算机视觉领域的一项基础任务，旨在估计相机在重访已知场景时的位姿。虽然基于点的分层重定位方法已展现出良好的可扩展性和效率，但它们通常受限于稀疏的图像观测和较弱的特征匹配。在本工作中，我们提出了SplatHLoc，一种新颖的分层视觉重定位框架，该框架使用特征高斯泼溅作为场景表示。为解决数据库图像的稀疏性问题，我们提出了一种自适应视点检索方法，该方法能合成与查询图像视点更接近的虚拟候选图像，从而提高了初始位姿估计的准确性。对于特征匹配，我们观察到高斯渲染的特征与直接从图像中提取的特征在两阶段匹配过程中表现出不同的优势：前者在粗匹配阶段表现更好，而后者在精匹配阶段被证明更为有效。因此，我们引入了一种混合特征匹配策略，以实现更准确、更高效的位姿估计。在室内和室外数据集上进行的大量实验表明，SplatHLoc增强了视觉重定位的鲁棒性，并创造了新的最佳性能水平。

## Abstract
Visual relocalization is a fundamental task in the field of 3D computer vision, estimating a camera's pose when it revisits a previously known scene. While point-based hierarchical relocalization methods have shown strong scalability and efficiency, they are often limited by sparse image observations and weak feature matching. In this work, we propose SplatHLoc, a novel hierarchical visual relocalization framework that uses Feature Gaussian Splatting as the scene representation. To address the sparsity of database images, we propose an adaptive viewpoint retrieval method that synthesizes virtual candidates with viewpoints more closely aligned with the query, thereby improving the accuracy of initial pose estimation. For feature matching, we observe that Gaussian-rendered features and those extracted directly from images exhibit different strengths across the two-stage matching process: the former performs better in the coarse stage, while the latter proves more effective in the fine stage. Therefore, we introduce a hybrid feature matching strategy, enabling more accurate and efficient pose estimation. Extensive experiments on both indoor and outdoor datasets show that SplatHLoc enhances the robustness of visual relocalization, setting a new state-of-the-art.

## 2. 方法论
*   **整体框架 (SplatHLoc)**：一个基于FGS的分层重定位框架（见图1,2）。流程分为离线的FGS建图和在线的分层重定位。在线阶段包含自适应视点检索和混合特征匹配两个核心模块。
*   **关键技术细节**：
    1.  **FGS场景表示与训练**：
        *   从SfM点云初始化高斯基元，每个基元包含位置、旋转、尺度、不透明度、颜色和特征向量。
        *   训练时联合优化辐射场和特征场。引入一个轻量级特征解码器 `d`，将渲染的低维特征图提升至高维，以减少存储和计算开销（见图3）。损失函数为光度损失（L1 + D-SSIM）和特征损失（L1）的加权和。
    2.  **自适应粗到细视点检索 (Adaptive C2F Viewpoint Retrieval)**：
        *   **粗检索**：使用VPR方法（如MixVPR）为查询图像检索Top-k1个候选数据库图像，并用轻量级稀疏匹配器进行几何验证，选择内点数最多的作为初始候选 `I_c^c`。
        *   **细检索（条件触发）**：若初始候选的内点数低于阈值 `I`，则在其位姿周围随机扰动（旋转±a°，平移±b米），生成k2个虚拟视点位姿，并用FGS渲染对应的彩色图。
        *   对这些虚拟视图再次进行检索和几何验证，选出与查询视点最接近的虚拟图像 `I_f^c` 作为最终参考图像。此方法旨在解决因数据库图像分布不均导致的视角差异问题。
    3.  **混合粗到细特征匹配 (Hybrid C2F Feature Matching)**：
        *   **关键观察**：FGS直接渲染的特征具有多视角知识且避免了从渲染图二次提取的误差累积，但与原查询图像提取的特征存在“特征间隙”，不利于精细几何建模。因此，前者更适合粗匹配，后者更适合精匹配。
        *   **粗匹配**：在低分辨率下计算查询特征图 `F_t` 与FGS渲染的高维特征图 `F_r^{high}` 之间的相似度矩阵。经过双向Softmax和互最近邻过滤得到粗对应关系 `C_{q,r}^c`。
        *   **精匹配**：使用预训练的密集/半密集匹配器（如JamMa）的编码器 `E_f` ，从查询图像 `I_q` 和渲染彩色图 `I_r` 中提取精细特征 `F_q^f, F_r^f`。以每个粗对应为中心裁剪局部窗口，计算精细相关性并解码得到亚像素级的精对应关系 `C_{q,r}^f`。
    4.  **位姿估计与迭代优化**：
        *   利用渲染的深度图将2D-2D对应 (`C_{q,r}^f`) 提升为2D-3D对应。
        *   使用RANSAC-PnP估计初始位姿。
        *   迭代优化：从当前估计位姿重新渲染彩色图和深度图，重复上述混合匹配与PnP步骤以细化位姿。
