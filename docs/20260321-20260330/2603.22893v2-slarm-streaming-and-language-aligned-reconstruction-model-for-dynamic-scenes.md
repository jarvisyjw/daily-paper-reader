## 摘要
我们提出了SLARM，一种前馈模型，它统一了动态场景重建、语义理解和实时流式推理。SLARM通过高阶运动建模捕捉复杂、非均匀的运动，仅基于可微分渲染进行训练，无需任何光流监督。此外，SLARM从LSeg中蒸馏语义特征以获得语言对齐的表征。这一设计使得能够通过自然语言进行语义查询，并且语义与几何之间的紧密耦合进一步增强了动态重建的准确性和鲁棒性。更重要的是，SLARM使用基于窗口的因果注意力处理图像序列，实现了稳定、低延迟的流式推理且不累积内存成本。在这一统一框架内，SLARM在动态估计、渲染质量和场景解析方面取得了最先进的结果，相较于现有方法将运动精度提高了21%，重建PSNR提高了1.6 dB，分割mIoU提高了20%。

## Abstract
We propose SLARM, a feed-forward model that unifies dynamic scene reconstruction, semantic understanding, and real-time streaming inference. SLARM captures complex, non-uniform motion through higher-order motion modeling, trained solely on differentiable renderings without any flow supervision. Besides, SLARM distills semantic features from LSeg to obtain language-aligned representations. This design enables semantic querying via natural language, and the tight coupling between semantics and geometry further enhances the accuracy and robustness of dynamic reconstruction. Moreover, SLARM processes image sequences using window-based causal attention, achieving stable, low-latency streaming inference without accumulating memory cost. Within this unified framework, SLARM achieves state-of-the-art results in dynamic estimation, rendering quality, and scene parsing, improving motion accuracy by 21%, reconstruction PSNR by 1.6 dB, and segmentation mIoU by 20% over existing methods.

#### **1. 核心问题与整体含义**
*   **研究背景**：神经辐射场（NeRF）和3D高斯溅射（3DGS）推动了静态3D重建的发展，但**动态场景**的重建仍面临挑战。现有方法多为优化型，耗时长且泛化性差；而前馈模型则多局限于静态场景。
*   **研究动机**：针对现有动态重建方法（如STORM）的三大局限提出解决方案：
    1.  **运动建模过于简化**：假设匀速运动，无法捕捉真实世界复杂的非均匀、非线性运动。
    2.  **功能单一**：仅关注几何重建，缺乏高级语义理解。
    3.  **推理效率低下**：需要批处理多帧输入，无法进行增量式的流式推理。
*   **整体含义**：SLARM旨在构建一个统一的、前馈式的4D高斯推理框架，一次性解决动态场景的**几何重建、语义理解和实时流式推理**三大核心任务。

#### **2. 方法论**
*   **核心思想**：构建一个基于Transformer的前馈模型，从带有相机位姿的视频序列中直接预测一个显式的4D高斯溅射表示。该表示包含随时间演化的几何、外观、运动矢量和语言对齐的语义特征。
*   **关键技术细节与流程**：
    1.  **高阶运动建模 (High-Order Motion Modeling)**：
        *   摒弃了简单的匀速或位移场假设。使用类似泰勒展开的多阶函数来建模位移 `Γ(Δt)`。
        *   网络为每个高斯点预测多个阶次（如L=3对应速度、加速度、急动度）的标量大小 `sl` 和方向向量 `vl`。总位移是各阶贡献在时间偏移 `Δt` 上的加权和。这使得模型能表达复杂的非刚性运动。
    2.  **语言对齐语义蒸馏 (Language-Aligned Semantics)**：
        *   从预训练的2D视觉-语言基础模型LSeg中蒸馏特征到4D高斯表示中。
        *   每个高斯点附带一个高维语义特征向量。通过alpha混合渲染出语义特征图，并与LSeg提取的特征计算MSE损失进行监督。
        *   对于有标注的数据，额外使用交叉熵损失将渲染特征与CLIP文本嵌入对齐，提升分类精度。
    3.  **流式推理架构 (Streaming Inference Architecture)**：
        *   采用基于窗口的因果注意力机制处理视频序列。每帧独立处理，同时通过一个紧凑的隐藏状态（记忆队列）传播历史信息。
        *   在在线模式下，遵循严格因果性：仅使用当前及过去帧的信息。通过将当前帧的动态高斯向后传播到最近的历史帧来“回溯”更新动态内容，而静态部分保持不变。这实现了恒定的延迟和内存消耗。
    4.  **统一多任务学习框架 (Unified Multi-Task Learning)**：
        *   模型主干为交替注意力Transformer（交替进行帧内注意力和全局注意力）。
        *   解码器包含三个并行的轻量级MLP头：`Gaussian Head`（预测位置、旋转、尺度等）、`Motion Head`（预测高阶运动系数）、`Semantic Head`（预测语义特征）。
        *   训练损失是多项任务的加权和：RGB渲染损失 (`Lrgb`, MSE+LPIPS)、深度一致性损失 (`Ldepth`)、天空区域正则化损失 (`Lsky`)、运动正则化损失 (`Lreg`)以及上述提到的语义蒸馏/分类损失 (`Lfeat = Lsem ∨ Lcls`) 。
