## 摘要
我们提出了SLARM，一种前馈模型，它统一了动态场景重建、语义理解和实时流式推理。SLARM通过高阶运动建模捕捉复杂、非均匀的运动，仅基于可微分渲染进行训练，无需任何光流监督。此外，SLARM从LSeg中蒸馏语义特征以获得语言对齐的表征。这一设计使得可以通过自然语言进行语义查询，并且语义与几何之间的紧密耦合进一步增强了动态重建的准确性和鲁棒性。更重要的是，SLARM使用基于窗口的因果注意力处理图像序列，实现了稳定、低延迟的流式推理且不累积内存成本。在此统一框架内，SLARM在动态估计、渲染质量和场景解析方面取得了最先进的结果，相较于现有方法，运动准确性提高了21%，重建PSNR提升了1.6 dB，分割mIoU提高了20%。

## Abstract
We propose SLARM, a feed-forward model that unifies dynamic scene reconstruction, semantic understanding, and real-time streaming inference. SLARM captures complex, non-uniform motion through higher-order motion modeling, trained solely on differentiable renderings without any flow supervision. Besides, SLARM distills semantic features from LSeg to obtain language-aligned representations. This design enables semantic querying via natural language, and the tight coupling between semantics and geometry further enhances the accuracy and robustness of dynamic reconstruction. Moreover, SLARM processes image sequences using window-based causal attention, achieving stable, low-latency streaming inference without accumulating memory cost. Within this unified framework, SLARM achieves state-of-the-art results in dynamic estimation, rendering quality, and scene parsing, improving motion accuracy by 21%, reconstruction PSNR by 1.6 dB, and segmentation mIoU by 20% over existing methods.

## SLARM：面向动态场景的流式与语言对齐重建模型 - 论文分析总结

### 1. 核心问题与整体含义
*   **研究背景**：神经辐射场（NeRF）和3D高斯泼溅（3DGS）推动了静态3D重建的发展。然而，现有动态场景重建方法通常存在以下问题：
    1.  **优化时间长**：需要数分钟到数小时进行逐场景优化，难以泛化。
    2.  **运动建模简单**：如STORM等方法假设匀速运动，无法捕捉真实世界复杂的非均匀、非线性运动。
    3.  **功能单一**：大多仅关注几何重建，缺乏高级语义理解能力。
    4.  **推理效率低**：需要批处理多帧输入或进行插值，无法实现增量式、低延迟的流式推理。
*   **研究动机**：为了克服上述局限，本文旨在提出一个统一的框架，能够同时实现**高质量动态重建**、**语言对齐的语义理解**和**实时流式推理**，以满足自动驾驶、具身智能等应用对实时感知与决策的需求。

### 2. 方法论
*   **核心思想**：构建一个前馈Transformer模型（SLARM），以视频序列和已知相机位姿为输入，直接输出一个显式的4D高斯泼溅表示。该表示包含随时间演化的几何、外观、运动以及语言对齐的语义特征。
*   **关键技术细节**：
    *   **高阶运动建模 (High-Order Motion Modeling)**：
        *   摒弃简单的匀速假设。将每个高斯的位移建模为时间的可微函数（泰勒展开）。
        *   网络预测速度标量 `sl` 和方向向量 `vl`（L2归一化），计算运动系数 `ml = sl · vl / ∥vl∥₂`。
        *   总位移 `Γ(Δt)` = Σ (ml · (Δt)^(l+1) / (l+1)!)，其中 l ∈ {0, ..., L-1}。实验采用 L=3（三阶），可建模速度、加速度和加加速度（jerk）。
        *   通过渲染监督学习运动：将时间 t 的高斯用预测的位移 `Γ(Δt)` 变换到 t+Δt，渲染得到图像后与真实帧计算RGB损失 (`Lrgb`)。
    *   **语言对齐语义蒸馏 (Language-Aligned Semantic Distillation)**：
        *   为每个高斯附加一个高维语义特征向量。
        *   从冻结的2D基础模型LSeg中提取每帧的2D语义特征作为教师信号。
        *   通过alpha混合渲染出时间 t+Δt 处的语义特征图 (`F̂t+Δt`)，并与LSeg提取的特征 (`F̃t+Δt`)计算MSE损失 (`Lsem`)进行蒸馏。
        *   对于有标注的数据，额外使用交叉熵损失 (`Lcls`)来对齐预测特征与CLIP文本嵌入。
    *   **流式推理架构 (Streaming Inference Architecture)**：
        *   采用基于窗口的因果注意力机制处理序列。每个新帧独立处理，同时传播一个紧凑的隐藏状态（记忆队列）。
        *   在在线模式下遵循严格因果性：仅利用当前及过去观测。动态高斯仅向后传播到最近的历史帧以进行细化更新。
        *   根据运动幅度阈值将高斯划分为静态和动态子集 (`Gstatic`, `Gdynamic`)，确保新时间步渲染无空洞并保持几何一致性。

### 7. 潜在局限性与未来工作
*   **计算资源需求**：训练需要64块NPU和4天时间，门槛较高。模型参数量未明确给出，但基于Transformer的架构通常较大。
