## 摘要
我们提出了ViewSplat，一种用于从无位姿图像进行新视角合成的视角自适应3D高斯泼溅网络。尽管近期的前馈式3D高斯泼溅技术通过绕过逐场景优化，显著加速了3D场景重建，但仍存在一个根本性的保真度差距。我们将此瓶颈归因于单步前馈网络在回归满足所有视角的静态高斯基元方面能力有限。为了解决这一限制，我们将范式从静态基元回归转向视角自适应动态泼溅。我们的流程学习的不是刚性的高斯表示，而是一种视角自适应的潜在表示。具体而言，ViewSplat首先预测基础高斯基元以及动态多层感知机的权重。在渲染过程中，这些多层感知机以目标视角坐标作为输入，并为每个高斯属性（即3D位置、尺度、旋转、不透明度和颜色）预测视角相关的残差更新。我们将此机制称为视角自适应动态泼溅，它允许每个基元修正初始估计误差，从而有效捕捉高保真度的外观。大量实验表明，ViewSplat在保持快速推理（17 FPS）和实时渲染（154 FPS）的同时，实现了最先进的保真度。

## Abstract
We present ViewSplat, a view-adaptive 3D Gaussian splatting network for novel view synthesis from unposed images. While recent feed-forward 3D Gaussian splatting has significantly accelerated 3D scene reconstruction by bypassing per-scene optimization, a fundamental fidelity gap remains. We attribute this bottleneck to the limited capacity of single-step feed-forward networks to regress static Gaussian primitives that satisfy all viewpoints. To address this limitation, we shift the paradigm from static primitive regression to view-adaptive dynamic splatting. Instead of a rigid Gaussian representation, our pipeline learns a view-adaptable latent representation. Specifically, ViewSplat initially predicts base Gaussian primitives alongside the weights of dynamic MLPs. During rendering, these MLPs take target view coordinates as input and predict view-dependent residual updates for each Gaussian attribute (i.e., 3D position, scale, rotation, opacity, and color). This mechanism, which we term view-adaptive dynamic splatting, allows each primitive to rectify initial estimation errors, effectively capturing high-fidelity appearances. Extensive experiments demonstrate that ViewSplat achieves state-of-the-art fidelity while maintaining fast inference (17 FPS) and real-time rendering (154 FPS).

*   **研究背景**：3D高斯泼溅（3DGS）已成为实现高保真实时渲染的强大3D表示方法。传统方法依赖逐场景优化，而近期研究转向**前馈式通用架构**，可直接从少量输入图像预测场景表示，实现快速重建和新视角合成。然而，这些方法通常严重依赖通过运动恢复结构（SfM）预先计算的精确相机位姿。
*   **核心瓶颈**：尽管出现了无需位姿的前馈方法（如SPFSplat），但其性能仍与优化方法存在显著差距。作者认为根本原因在于：单步前馈网络难以回归出能满足所有潜在视角的**静态高斯图元**。对于复杂几何或非朗伯表面，一次性预测最优属性是一个不适定问题。
*   **研究动机**：为了解决上述“静态图元瓶颈”，本文提出将范式从静态图元回归转变为**视角自适应动态泼溅**。核心思想是：与其费力预测通用的3D高斯，不如让网络专注于生成更容易学习的、针对特定视角的更新。

## 2. 提出的方法论
*   **核心思想**：ViewSplat引入了一种动态机制，能够为每个目标视角即时更新高斯图元。模型预测两部分内容：
    1.  **基础高斯基元集**
    2.  **动态多层感知机（MLP）的权重**
    *   这些动态MLP以输入场景上下文为条件生成其参数。
    *   在渲染阶段，这些MLP以目标视角坐标作为输入，为所有高斯属性（位置、尺度、旋转、不透明度、球谐系数）预测**视角相关的残差更新**。
    *   将这些残差加到基础高斯基元上，即可得到针对该视角优化的最终高斯参数。

*   **关键技术细节**：
    1.  **整体架构基础**：基于自监督、无需位姿的前馈框架SPFSplat及其变体构建。使用几何Transformer骨干网络（如MASt3R, VGGT）提取多视图特征。
    2.  **规范空间重建与位姿估计**：
        *   两个DPT头分别预测规范空间中的高斯中心位置和其他属性。
        *   一个位姿头估计输入视图到规范参考视图的相对变换矩阵。
    3.  **视角相关头与MLP生成器 (View-Dependent Head)**：
        *   这是一个基于DPT的超网络架构。它接收几何Transformer的特征令牌，并动态生成像素级的“视图MLP”的参数。
        *   这种设计允许模型根据场景局部特性生成特定的响应函数。
    4.  **目标位姿重参数化 (Target Pose Reparameterization)**：
        *   为避免直接将复杂的变换矩阵输入MLP导致优化困难，将目标外参矩阵重参数化为一个4维向量：[单位视线方向 (3维), 对数距离 (1维)]。
        *   `距离 = log(||相机中心 - 高斯中心|| + ε)`。对数变换能压缩距离的动态范围，使网络对近场变化更敏感。
    5.  **损失函数**: `L_total = L_render + λ_reproj L_reproj`
        *   `L_render`: MSE光度损失 + LPIPS感知损失组合的图像渲染损失。
        *   `L_reproj`: 利用估计的相对位姿将预测的3D高斯中心投影回2D图像平面并与原始像素坐标对齐的重投影损失。
