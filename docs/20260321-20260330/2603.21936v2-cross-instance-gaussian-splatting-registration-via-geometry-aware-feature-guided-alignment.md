## 摘要
我们提出了高斯泼溅对齐（GSA），一种通过相似变换（旋转、平移和缩放）对齐两个独立3D高斯泼溅（3DGS）模型的新方法，即使它们是同一类别中的不同对象（例如，不同的汽车）。相比之下，现有方法只能对齐同一对象的3DGS模型（例如，同一辆汽车），并且通常必须将真实尺度作为输入，而我们能成功估计它。GSA利用视点引导的球形图特征来获取鲁棒的对应关系，并引入了一个两步优化框架，在保持模型固定的同时对它们进行对齐。首先，我们应用迭代的特征引导绝对定向求解器作为粗配准，它对较差的初始化具有鲁棒性（例如，180度错位或10倍的尺度差距）。接着，我们使用一个精细配准步骤来强制多视图特征一致性，其灵感来源于逆辐射场公式。第一步已经达到了最先进的性能，第二步则进一步改善了结果。在相同对象的情况下，GSA优于先前的工作，通常优势显著，即使其他方法被提供了真实尺度。在更困难的同一类别不同对象的情况下，GSA大大超越了它们，为类别级3DGS配准提供了首个有效的解决方案并解锁了新的应用。项目网页：https://bgu-cs-vil.github.io/GSA-project/

## Abstract
We present Gaussian Splatting Alignment (GSA), a novel method for aligning two independent 3D Gaussian Splatting (3DGS) models via a similarity transformation (rotation, translation, and scale), even when they are of different objects in the same category (e.g., different cars). In contrast, existing methods can only align 3DGS models of the same object (e.g., the same car) and often must be given true scale as input, while we estimate it successfully. GSA leverages viewpoint-guided spherical map features to obtain robust correspondences and introduces a two-step optimization framework that aligns 3DGS models while keeping them fixed. First, we apply an iterative feature-guided absolute orientation solver as our coarse registration, which is robust to poor initialization (e.g., 180 degrees misalignment or a 10x scale gap). Next, we use a fine registration step that enforces multi-view feature consistency, inspired by inverse radiance-field formulations. The first step already achieves state-of-the-art performance, and the second further improves results. In the same-object case, GSA outperforms prior works, often by a large margin, even when the other methods are given the true scale. In the harder case of different objects in the same category, GSA vastly surpasses them, providing the first effective solution for category-level 3DGS registration and unlocking new applications. #### **1. 核心问题与整体含义**
*   **研究背景**：3D高斯泼溅（3DGS）已成为高效、高保真新视角合成的主流表示方法。然而，对齐两个独立重建的3DGS模型是一个开放挑战。现有方法（如基于点云配准或专门为3DGS设计的GaussReg）通常只能处理**同一物体**的重建模型对齐，且往往需要已知真实尺度。
*   **研究动机**：本文旨在解决更具挑战性的**类别级跨实例配准**问题，即对齐同一类别下不同物体的3DGS模型（例如，两辆不同的汽车）。这面临三大难点：1) 物体间存在形状、尺度和外观差异；2) 由于基于运动恢复结构（SfM）的重建过程，模型的尺度、位置和方向是任意的；3) 现有方法对初始化敏感且在几何差异下容易失败。
*   **整体含义**：本文提出了首个能够有效解决类别级3DGS配准的方法（GSA），成功估计了包含旋转、平移和缩放的相似变换，解锁了如几何一致的对象替换、同步多视角合成等新应用。

#### **2. 方法论**
*   **核心思想**：采用一个“粗配准 + 细配准”的两步优化框架。关键在于利用具有几何感知能力的语义特征来引导对应关系搜索，从而克服跨实例间的几何差异和初始化难题。
*   **关键技术细节**：
    1.  **预处理与特征增强模型构建**：
        *   使用COLMAP估计相机位姿，SAM生成物体掩码。
        *   采用Mariotti等人[21]提出的“视点引导球面图特征”，该特征能区分左右、前后等空间方位（见图4），优于DINOv2等纯语义特征。
        *   在构建3DGS模型时，除了优化颜色和几何外，还将上述2D特征“提升”到每个高斯的属性中，得到同时包含颜色和几何感知特征的增强型3DGS模型。
    2.  **粗配准（算法3）**：
        *   将两个增强后的3DGS模型视为带有特征描述符的点云。
        *   迭代执行改进的ICP流程：对于源模型的每个点，首先根据**特征相似性阈值τ_f**在目标模型中筛选候选点集（打破纯空间最近邻的限制），然后在该候选集中寻找空间最近点作为对应点。
        *   使用结合Kabsch-Umeyama（求旋转/平移）和Horn对称尺度公式的混合闭式解法来求解最优相似变换。
        *   此步骤对极差的初始化（如180°旋转、10倍尺度差）鲁棒。
    3.  **细配准（算法4）**：
        *   将粗配准结果作为初始值。
        *   受逆辐射场方法启发，提出“多视图特征一致性损失”：固定两个3DGS模型和一组目标相机位姿，通过可微渲染分别渲染两个模型的2D特征图，并最小化它们之间的L2距离总和（公式8）。
        *   此步骤通过梯度下降优化变换参数T ∈ Sim(3)，利用渲染过程直接约束多视角下的语义-几何对齐一致性。

当然，该方法也存在局限性，特别是对预训练视觉特征的依赖以及在复杂场景中未经测试。然而，这些并未削弱其核心贡献的价值，反而为后续研究指明了清晰的改进方向。总体而言，这篇论文为3DGS社区提供了一个强大且实用的新工具，并有望推动相关应用的发展。
