## 摘要
从非结构化、真实世界的图像中重建大规模场景的精确三维模型，仍然是计算机视觉领域的核心挑战，尤其是在输入视图之间几乎没有或完全没有重叠的情况下。在这种情况下，现有的重建流程常常会产生多个互不连接的部分重建结果，或者错误地将非重叠区域合并为重叠的几何结构。在本研究中，我们提出了一个框架，将每个部分重建结果与场景的完整参考模型进行关联（grounding），从而即使在缺乏视觉重叠的情况下也能实现全局一致的对齐。我们从谷歌地球工作室生成的密集、地理空间精确的伪合成渲染图中获取参考模型。这些渲染图提供了完整的场景覆盖，但其外观与现实世界的照片存在显著差异。我们的核心见解是，尽管存在这种显著的领域差异（domain gap），但两个领域共享相同的基础场景语义。我们使用三维高斯泼溅来表示参考模型，并为每个高斯分布添加语义特征，并将对齐问题表述为一个基于逆向特征的优化方案：在保持参考模型固定的同时，估计一个全局六自由度位姿和尺度。此外，我们引入了WikiEarth数据集，该数据集将现有的部分三维重建结果与伪合成参考模型进行了配准。我们证明，当使用各种经典和基于学习的流程进行初始化时，我们的方法能持续改进全局对齐效果，同时缓解了最先进的端到端模型的失败模式。所有代码和数据都将发布。

## Abstract
Reconstructing accurate 3D models of large-scale real-world scenes from unstructured, in-the-wild imagery remains a core challenge in computer vision, especially when the input views have little or no overlap. In such cases, existing reconstruction pipelines often produce multiple disconnected partial reconstructions or erroneously merge non-overlapping regions into overlapping geometry. In this work, we propose a framework that grounds each partial reconstruction to a complete reference model of the scene, enabling globally consistent alignment even in the absence of visual overlap. We obtain reference models from dense, geospatially accurate pseudo-synthetic renderings derived from Google Earth Studio. These renderings provide full scene coverage but differ substantially in appearance from real-world photographs. Our key insight is that, despite this significant domain gap, both domains share the same underlying scene semantics. We represent the reference model using 3D Gaussian Splatting, augmenting each Gaussian with semantic features, and formulate alignment as an inverse feature-based optimization scheme that estimates a global 6DoF pose and scale while keeping the reference model fixed. Furthermore, we introduce the WikiEarth dataset, which registers existing partial 3D reconstructions with pseudo-synthetic reference models. We demonstrate that our approach consistently improves global alignment when initialized with various classical and learning-based pipelines, while mitigating failure modes of state-of-the-art end-to-end models. All code and data will be released.

*   **关键洞察**：尽管存在巨大的外观差异，但真实照片和伪合成渲染图共享相同的底层**场景语义**。利用语义特征而非颜色信息，可以跨越领域鸿沟实现鲁棒的对齐。

## 2. 方法论
*   **整体流程**：
    1.  **参考模型构建**：使用谷歌地球工作室渲染密集视图，通过COLMAP重建出完整的3D参考模型。
    2.  **参考模型表示与增强**：使用3D高斯泼溅来表示参考模型。关键创新在于为每个高斯分布蒸馏并附加了DINOv2提取的语义特征向量（借鉴Feature 3DGS方法），形成“语义化”的3DGS。
    3.  **部分重建对齐（逆向优化）**：
        *   输入：一个由SfM预先捆绑好的部分图像集合（称为“元图像”）及其初始粗略位姿。
        *   目标：估计一个全局的6自由度+尺度的变换矩阵 `T`，将该元图像对齐到固定的参考模型上。
        *   优化过程：
            a. 使用当前估计的 `T`，从参考模型中渲染出对应视角的语义特征图。
            b. 计算渲染特征与元图像中真实图像提取的DINOv2特征之间的L1损失 `L_sem`。
            c. 通过反向传播仅更新变换参数 `T`（参考模型的3DGS参数被冻结）。
    4.  **鲁棒优化机制**：为了处理真实图像中的遮挡物和参考模型中的漂浮物等异常值，采用了最小截断二乘法。在每次迭代中，忽略那些损失值高于上一轮损失中位数的图像（即“软选择”机制），确保优化不被离群点带偏。

*   **新基准数据集 WikiEarth**：
    *   **目的**：为评估跨域三维对齐方法提供标准化基准。
    *   **构成**：将来自WikiScenes数据集的32个部分三维重建（覆盖23个不同地标）与从谷歌地球工作室构建的伪合成参考模型进行精确配准。每个部分重建包含8-713张不等的互联网图片。
    *   **真值获取**：通过手动调整COLMAP参数并进行视觉检查过滤来获得高质量的配准真值。

    *   **机器人视觉与SLAM**：为机器人在未知但具有部分先验地理信息（如粗糙建筑轮廓）的环境中的长期定位与建图提供新思路。
3.  **未来研究方向**：
    *   **减少对外部参考的依赖**：探索如何从互联网图像集合自身推断或生成轻量级的全局语义/几何先验，以降低对谷歌地球等特定数据源的依赖。
    *   **改进初始化鲁棒性**：结合基于学习的全局位姿估计方法或更强大的特征匹配技术，为优化提供更可靠的初始值，扩大方法的收敛域。
    *   **处理动态场景与遮挡**：进一步增强语义特征的判别力与鲁棒性，以更好地区分场景的静态结构和动态遮挡物（如行人、车辆）。
    *   **扩展到更广泛场景**：验证方法在室内环境、自然景观或其他缺乏高质量合成模型的场景下的适用性与局限性。

## 10. 总结
《真实世界中的场景关联》一文针对从稀疏、非重叠的互联网图像进行大规模三维重建的核心难题，提出了一个创新且实用的解决方案。其核心贡献在于引入了“场景关联”的概念，即利用一个全局一致的伪合成参考模型作为锚点，通过基于DINOv2语义特征的逆向优化，将零散的部分重建对齐到统一的坐标系中。该方法巧妙地利用了语义信息跨越了真实照片与低质量渲染图之间的领域鸿沟。

论文通过构建全面的WikiEarth基准数据集、设计严谨的实验以及与多种基线及前沿模型的对比分析，充分证明了所提方法的有效性和优越性。尽管存在对初始化和外部参考模型的依赖等局限性，但这项工作为处理“野外”图像的三维理解开辟了一条新的道路，在文化遗产保护、AR和机器人等领域具有明确的应用前景。其实验充分性、技术实现质量和开源精神使其成为该领域一项扎实且重要的研究。
