## 摘要
我们提出了SLARM，一种前馈模型，它统一了动态场景重建、语义理解和实时流式推理。SLARM通过高阶运动建模捕捉复杂、非均匀的运动，仅基于可微分渲染进行训练，无需任何光流监督。此外，SLARM从LSeg中蒸馏语义特征以获得语言对齐的表征。这一设计使得能够通过自然语言进行语义查询，并且语义与几何之间的紧密耦合进一步增强了动态重建的准确性和鲁棒性。更重要的是，SLARM使用基于窗口的因果注意力处理图像序列，实现了稳定、低延迟的流式推理且不累积内存成本。在此统一框架下，SLARM在动态估计、渲染质量和场景解析方面取得了最先进的结果，相较于现有方法将运动精度提高了21%，重建PSNR提升了1.6 dB，分割mIoU提高了20%。

## Abstract
We propose SLARM, a feed-forward model that unifies dynamic scene reconstruction, semantic understanding, and real-time streaming inference. SLARM captures complex, non-uniform motion through higher-order motion modeling, trained solely on differentiable renderings without any flow supervision. Besides, SLARM distills semantic features from LSeg to obtain language-aligned representations. This design enables semantic querying via natural language, and the tight coupling between semantics and geometry further enhances the accuracy and robustness of dynamic reconstruction. Moreover, SLARM processes image sequences using window-based causal attention, achieving stable, low-latency streaming inference without accumulating memory cost. Within this unified framework, SLARM achieves state-of-the-art results in dynamic estimation, rendering quality, and scene parsing, improving motion accuracy by 21%, reconstruction PSNR by 1.6 dB, and segmentation mIoU by 20% over existing methods.

## 2. 方法论
*   **核心思想**：SLARM是一个基于4D高斯溅射的统一前馈Transformer模型。它维护一个随时间演化的显式4D高斯表示，该表示同时编码几何、外观、运动和语义信息。
*   **关键技术细节与流程**：
    1.  **输入与特征提取**：给定视频序列及相机内外参，使用ViT提取图像特征，并融入Plücker光线坐标和绝对时间戳等几何与时序先验。
    2.  **交替注意力Transformer骨干网络**：采用帧内注意力和全局注意力交替的机制，有效捕捉时空结构。
    3.  **高阶运动建模（核心创新）**：
        *   摒弃了简单的匀速或位移场假设。将位移建模为时间的可微函数，使用多阶泰勒展开（论文中采用三阶）。
        *   网络为每个高斯预测各阶的标量速度和归一化的方向向量 `ml = sl · vl / ∥vl∥₂`。
        *   总位移通过聚合各阶贡献计算：`Γ(Δt) = Σ_{l=0}^{L-1} ml · (Δt)^{l+1} / (l+1)!`。这显式地建模了位置对时间的前三阶导数（速度、加速度、加加速度），能更好地捕捉复杂非刚性运动。

    5.  **语言对齐语义蒸馏**：
        *   为每个高斯附加一个高维语义特征向量。
        *   从冻结的2D基础模型LSeg中提取每帧的2D语义特征作为教师信号。
        *   通过将变形后高斯的语义属性进行alpha混合渲染出语义特征图，并与LSeg特征计算MSE损失进行知识蒸馏。对于有标注的数据，额外使用交叉熵损失对齐CLIP文本嵌入。
    6.  **流式推理架构（核心创新）**：
        *   采用基于窗口的因果注意力机制，仅依赖当前及过去观测帧进行推理，满足实时性要求。
        *   将高斯根据运动幅度阈值划分为静态和动态子集。在在线模式下，动态高斯仅向后传播到最近的历史帧以合成目标视图，避免了渲染空洞并保持恒定内存和延迟。
