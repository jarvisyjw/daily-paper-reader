## 摘要
并行计算与GPU加速的最新进展为计算密集型学习问题（如主动SLAM——通过选择动作来降低不确定性并改进联合建图与定位）创造了新的机遇。然而，现有的基于深度强化学习的方法仍受限于缺乏可扩展的并行训练。在本工作中，我们通过提出一个可扩展的端到端深度强化学习框架来解决这一挑战，该框架支持大规模并行训练用于主动SLAM。与现有技术相比，我们的方法显著减少了训练时间，支持连续动作空间，并促进了对更真实场景的探索。该框架已作为开源项目发布，以促进可复现性和社区采用。

## Abstract
Recent advances in parallel computing and GPU acceleration have created new opportunities for computation-intensive learning problems such as Active SLAM -- where actions are selected to reduce uncertainty and improve joint mapping and localization. However, existing DRL-based approaches remain constrained by the lack of scalable parallel training. In this work, we address this challenge by proposing a scalable end-to-end DRL framework for Active SLAM that enables massively parallel training. Compared with the state of the art, our method significantly reduces training time, supports continuous action spaces and facilitates the exploration of more realistic scenarios. It is released as an open-source framework to promote reproducibility and community adoption.

## 2. 提出的方法论
*   **核心思想**：构建一个完全GPU加速、可大规模并行的DRL训练流水线。通过同时模拟数百个机器人智能体（如图1所示），在NVIDIA Isaac Sim/Lab仿真器中并行收集经验数据，从而将数天的训练时间压缩至数小时。
*   **关键技术细节**：
    *   **GPU向量化固定滞后SLAM骨干网络**：设计了一个轻量级的固定滞后位姿图优化器作为SLAM后端。它舍弃了全局闭环检测以换取确定性的、恒定时间的计算开销。该后端完全在GPU上以批处理和向量化方式运行，能够为数百个并行环境实时提供位姿估计及其协方差矩阵。
    *   **不确定性感知的观察与奖励设计**：
        *   **观察空间 (ot)**：包含128维激光雷达测距值、机器人偏航角(yaw)以及一个从SLAM协方差矩阵提取的标量化不确定性信号(Ut)。
        *   **奖励函数 (Rt)**：精心设计的多目标奖励函数，平衡了三个关键目标：
            1.  **探索奖励 (Rexplore)**：鼓励访问新区域。其激励强度由一个内在缩放因子(I)动态调节，该因子结合了当前不确定性水平(ct-1)和瞬时信息增益(ιt)，实现了在低不确定性时鼓励探索、在高不确定性时鼓励降低不确定性的平滑切换。
            2.  **绝对不确定性惩罚 (Rabs)**：惩罚处于高不确定性状态的操作。
            3.  **微分不确定性奖励 (R∆)**：奖励连续时间步间不确定性的降低。
        *   此外，引入了“反贴墙行走”正则化项(αhug)，以防止智能体学习到紧贴墙壁移动的次优策略。
    *   **策略优化与架构**：采用近端策略优化算法（PPO）进行策略优化。网络架构包含一个GRU循环层（隐藏层大小256），使智能体能够捕捉不确定性的时序动态（如上升或下降趋势），这对于做出有效的重定位决策至关重要。
    *   **SLAM无关的微调桥接器**：为了解决从训练用的轻量级SLAM后端切换到实际部署中更复杂的ROS2 SLAM系统（如slam_toolbox）时可能出现的观测分布偏移问题，提出了一个“训练桥接器”。它通过线性插值的方式，在微调阶段逐步将新SLAM系统的不确定性信号混合到观察中，确保策略平稳适应。
