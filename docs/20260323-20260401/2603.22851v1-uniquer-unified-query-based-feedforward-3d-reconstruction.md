## 摘要
我们提出了UniQueR，一个统一的基于查询的前馈框架，用于从无位姿图像中进行高效且准确的三维重建。现有的前馈模型（如DUSt3R、VGGT和AnySplat）通常预测逐像素的点云图或像素对齐的高斯分布，这些方法本质上仍是2.5D的，且仅限于可见表面。相比之下，UniQueR将重建问题表述为一个稀疏的三维查询推断问题。我们的模型学习一组紧凑的三维锚点作为显式的几何查询，使网络能够在单次前向传播中推断场景结构（包括遮挡区域的几何信息）。每个查询直接在全局三维空间（而非逐帧相机空间）中编码空间和外观先验，并生成一组用于可微分渲染的三维高斯分布。通过利用跨多视图特征的统一查询交互以及解耦的交叉注意力设计，UniQueR在显著降低内存和计算成本的同时，实现了强大的几何表达能力。在Mip-NeRF 360和VR-NeRF数据集上的实验表明，UniQueR在渲染质量和几何精度上均超越了最先进的前馈方法，且使用的基元数量比密集方法少一个数量级。

## Abstract
We present UniQueR, a unified query-based feedforward framework for efficient and accurate 3D reconstruction from unposed images. Existing feedforward models such as DUSt3R, VGGT, and AnySplat typically predict per-pixel point maps or pixel-aligned Gaussians, which remain fundamentally 2.5D and limited to visible surfaces. In contrast, UniQueR formulates reconstruction as a sparse 3D query inference problem. Our model learns a compact set of 3D anchor points that act as explicit geometric queries, enabling the network to infer scene structure, including geometry in occluded regions--in a single forward pass. Each query encodes spatial and appearance priors directly in global 3D space (instead of per-frame camera space) and spawns a set of 3D Gaussians for differentiable rendering. By leveraging unified query interactions across multi-view features and a decoupled cross-attention design, UniQueR achieves strong geometric expressiveness while substantially reducing memory and computational cost. Experiments on Mip-NeRF 360 and VR-NeRF demonstrate that UniQueR surpasses state-of-the-art feedforward methods in both rendering quality and geometric accuracy, using an order of magnitude fewer primitives than dense alternatives.

## 2. 方法论
*   **核心思想**：使用一组稀疏的、可学习的3D查询作为场景的全局锚点。每个查询关联一个3D位置和潜在嵌入，并通过交叉注意力机制与多视图图像特征交互。最终，每个查询“生成”一组3D高斯分布用于可微分渲染。
*   **关键技术细节**：
    1.  **网络流程**：
        *   **图像编码与几何先验预测**：使用DINOv2 ViT提取图像特征，并通过交替注意力Transformer聚合跨视图信息。解码器同时预测相机位姿、点云图和置信度图。
        *   **查询初始化与传播**：
            *   采用混合初始化策略：一半查询从预测的点云图中采样（对齐观测表面），另一半在3D空间中均匀采样（探索未观测区域）。
            *   设计了高效的解耦注意力模块：首先进行“查询→图像”的交叉注意力以吸收特征；然后进行“查询→查询”的自注意力以整合信息。这比将所有token拼接后进行全自注意力更高效。
    2.  **高斯生成与渲染监督**：
        *   每个更新后的查询根据其潜在嵌入预测一个位置偏移和K个高斯分布的属性（中心、尺度、旋转、不透明度、颜色）。

## 7. 优点
1.  **(核心创新)**提出了基于全局三维查询的前馈重建范式。该方法从根本上解决了像素对齐表示的局限性(**view-anchored limitation**) ，能够显式地建模遮挡区域和未观测空间。
2. **(技术创新)**设计了高效的解耦交叉注意力机制(`Decoupled Cross-Attention`)，显著降低了计算复杂度(`O(QN HW/p² + Q²)` vs `O((Q+N HW/p²)²)` ) ，使模型能够扩展到更多输入视图和高分辨率图像。
3. **(训练策略)**引入了混合查询初始化和新视角监督策略。前者稳定了训练过程并确保了对可见/不可见区域的覆盖；后者强制模型学习完整的场景表示而非仅拟合输入视图。

##8 .不足与局限
1 .**(应用限制)**当前框架不支持动态场景的重建 。论文指出将其扩展到包含时间动态的场景是一个有前景的未来方向 。
2 .**(性能权衡)**在密集视图设定下 ，仅使用前馈模式时 ，由于UniQueR使用了非常稀疏的基元(~262K Gaussians) ，其直接渲染质量有时会略逊于生成密集基元的像素对齐方法 。这表明其在极高保真度需求下的纯前馈性能存在权衡 。
3 .**(依赖预训练组件)**模型的图像编码器和点云图解码器部分是从Pi³等现有工作中初始化的并保持冻结 。这在一定程度上限制了端到端优化的潜力 ，且性能可能受限于这些预训练组件的泛化能力 。
4 .**(数据偏差风险)**尽管使用了大规模混合数据集进行训练 ，但模型的泛化能力仍需在更广泛 、更具挑战性的“野外”环境中进一步验证 。
